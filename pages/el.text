h2. Email <small>https://mail.riseup.net</small>

Προσφέρουμε ασφαλείς και ιδιωτικούς λογαριασμούς email, προσβάσιμους μέσω web, shell, IMAP, ή POP. 
* [[Βρείτε βοήθεια για τα email => email]]. 
* "Συνδεθείτε στο webmail":https://mail.riseup.net. 
* "Ζητήστε έναν λογαριασμό":https://user.riseup.net/forms/new_user. 
* "Επαναφορά χαμένου κωδικού":https://user.riseup.net/forms/password_reset/first. 
* "Αλλαγή των ρυθμίσεών μου":https://user.riseup.net.

h2. Λίστες email  <small>https://lists.riseup.net</small>

Προσφέρουμε λίστες email για οργανώσεις ακτιβιστών. 
* [[Βρείτε βοήθεια για τις λίστες => lists]]. 
* "Συνδεθείτε στις λίστες":https://lists.riseup.net/www/. 
* [[Κάντε αίτηση για λίστα =>  creating-lists]]. 
* [[Γίνετε συνδρομητές => subscribing]]. 
* [[Διαγραφείτε από συνδρομητές => subscribing]].

h2. Νέες Υπηρεσίες

Δοκιμάζουμε υπηρεσίες [*[[VPN]]*], [*[[chat]]*], και [*"etherpad":https://pad.riseup.net*]. Αυτές οι υπηρεσίες είναι ακόμα υπό ανάπτυξη και πιθανώς είναι αναξιόπιστες μερικές φορές. Άν θέλετε να βοηθήσετε να τις δοκιμάσουμε, δώστε τους μια ευκαιρία.
